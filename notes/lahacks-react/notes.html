React is a framework that allows the creation of your own "html" tags. These are called components, which are usable.

Before we dive in, know what a class is in Javascript. You can read about it in their README on their github. Their README is actually very detailed as well, so make sure you give it a read!

To start, you want to type in the following command, npx create-react-app [directory] which will do all the setup for you to make a React app.

From here, you then type npm start to essentially run the app that is built.

Props are attributes passed in from the parent component to the child component, similar to inherited attributes.

The map function allows to "map" each part of a list to a particular object or other thing.
