Notes from Lesson 34: The Form Tag

The form tag is a container for inputs (which basically makes the whole container a block instead of inline). Note that, as discussed before, there are two tags:

  • action - where the form sends data to
  • method - what HTTP method (GET/POST)
If action is not defined, it defaults to the same page. If method is not defined, it defaults to a GET request.
