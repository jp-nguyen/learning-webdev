Notes from Lesson 28: HTML Tables

To make a table, you use the following tags:

  • table - start of a table
  • tr - a row of a table
  • td - a column of a table
  • th - a header column of a table

Name Age
Rusty 2
Wyatt 13

There is also thead and tbody which do the same thing on the table:

Name Age
Rusty 2
Wyatt 13

These accomplish the same but are more readable.
