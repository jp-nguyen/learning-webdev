Notes from Lesson 25: HTML Attributes

Attributes are a way to add additional information to a tag. They come in a key-valuepair in tags, usually formatted as a ``attribute = string value''. We will cover only some attributes.

Images

These tags usually don't have much stuff. They have an attribute called src that is equal to the url of the image. This can be from another host or part of the machine. The img tag is a self-closing tag. For instance, this picture of Kirby:

is done with the following text:

<img src="https://upload.wikimedia.org/wikipedia/en/5/5c/Kirby.png">

Links

The a tag (which means anchor) has the attribute href which is set to the url. For instance, here is a link to Google.
