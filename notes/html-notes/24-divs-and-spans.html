Notes from Lesson 24: Divs and Spans

divs and spans are HTML tags that are not as useful until we get into CSS styling.

A div is a generic container for content. This basically allows you to group things together in HTML. For instance, if you want to group a heading and paragraph together, you can use a div tag to do so and add some styling to the background.

A span is also a generic container for content, but there is a key difference: the level of the tags. While a div is a block-level tag, a span is an inline-level tag.
