Notes from Lesson 30: Forms

For forms, there are three main tags that are going to be covered:

  • form tag
  • input tag
  • label tag
We will also cover input validation for this. Note that most of these tags won't necessary work until we have a backend to do this.

The form tag is a container that holds the interactive elements (e.g. inputs and buttons). This tag has two important attributes:

  • action - the URL to send form data to
  • method - the type of HTTP request

The input tag creates the interactive controls, which is determined by the type of input. There's around 30-40 types, but there are common types that are used all the time.
