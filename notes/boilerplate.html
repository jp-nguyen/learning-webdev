Notes from Lesson 19: HTML Boilerplate and Comments

You can set up the HTML boilerplate in VS Code by first typing in "html" on the file, then tabbing the second option to create the whole template! That's some litty-spaghitty stuff.

Having a <!DOCTYPE html> means you are declaring the file as an HTML file.

The <head> element provides general information about the document, including title and links to scripts and style sheets. Everything unseen usually goes in here.

In contrast, the <body> element provides the content of the document, i.e. everything that is seen.

You can toggle commands in VS Code by typing in Command+Slash (⌘/) on the line that you want to comment out. Comments usually look something like this: <!-- This is a comment! -->.
