HTML is a markup used to tell the browser how to structure web pages you visit.

In general, everything in HTML is an element, enclosed with an opening tag and a closing tag (both of which usually have angle brackets). Some may not have the closing tag though, for example img.

There are two types of elements:

  1. block-level: appearing on a new line
  2. inline-level: appearing on the same line
These are important categories to know!

Elements can also have attributes, which add more information that you don't want to show in the content. An attribute has the following:

  • a space between it and the element name/previous attribute
  • the attribute name followed by equals
  • the attribute value
For instance, this hyperlink to a great music video has three attributes!

Note: Some attributes may not have values (they act as booleans). In addition, try to always quote the attribute values. It's just better practice.

There are some characters that need escaping to be shown. For instance, < and > are literally typed as &lt and &gt, respectively.
